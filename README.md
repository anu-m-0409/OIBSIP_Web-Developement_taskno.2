# OIBSIP_Web-Developement_taskno.2
This is a tribute page.

 # Overview

The Tribute Page is a simple, single-page web project dedicated to Dr. A. P. J. Abdul Kalam, one of India’s most respected scientists and former Presidents.
The page highlights his life, achievements, and inspirational thoughts in a clean and minimal layout.

This project is created using HTML and CSS only, focusing on clarity, readability, and professional presentation.

# Features

Single-page tribute layout

Clean and minimal design

Inspirational quote section

Brief biography and contributions

Professional black-and-white themed styling

# Technologies Used

HTML5 – Structure and content

CSS3 – Styling and layout design
# Learning Outcome

Structuring content using HTML

Designing clean layouts using CSS

Creating professional one-page websites

Understanding typography and spacing for readability

# Advantages

Simple and easy to understand

Beginner-friendly code

No JavaScript or backend required

Suitable for academic and professional showcase
